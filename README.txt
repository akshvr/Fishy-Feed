Fishy Feed

This version preserves the original Fish Guardian app behavior, with only:
1. App name changed to Fishy Feed
2. Optional Micro:bit Web Serial alert support added

Files:
- index.html
- sketch.js
- microbit_makecode.js

How to use:
1. In p5.js Web Editor, replace index.html and sketch.js with these files.
2. Run the sketch and allow webcam access.
3. Optional Micro:bit:
   - Open MakeCode
   - Switch to JavaScript
   - Paste microbit_makecode.js
   - Download to Micro:bit
   - Keep Micro:bit connected by USB
   - In Chrome/Chromium, click "Connect Micro:bit"
4. The browser app still works even when no Micro:bit is connected.
